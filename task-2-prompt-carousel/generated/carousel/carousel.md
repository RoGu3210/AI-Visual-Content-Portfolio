## How to publish the carousel
1. Open each link in order (0 → 4).
2. In Excalidraw: **Menu → Export image** → choose **PNG** (or SVG), background on,
   2× scale for crisp social images. Export all four at the same size.
3. Upload the four PNGs in order to your carousel (LinkedIn / Instagram / X).
